## 자료구조 기초

- 자료구조란?
  - 컴퓨터에서 여러 자료들을 조직적, 체계적으로 저장하는 방법.
  - 보통 동일한 자료형을 저장하는 구조를 의미한다.
  - 자료구조에 따라 요소들 사이의 관계를 정의하는 규칙이 있다.
- 자료구조의 효율성
  - 효율성은 주로 시간 복잡도(time complexity)를 말한다.
  - 공간 복잡도(space complexity)를 포함하는 경우도 있지만 최근에는 하드웨어의 발전으로 인해 그렇게 중요하지 않다.
